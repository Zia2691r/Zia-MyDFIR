# Projects

---

## 1. Exploration for CipherText Visuals in Unsecured Cloud Computing

Description:  
A cloud security research project focused on protecting visual data stored in unsecured cloud environments using encryption and searchable ciphertext techniques.

Technologies Used:
- Java  
- AES Encryption  
- SHA-256  
- MySQL  
- Cloud Simulation  

Objective:
To secure sensitive cloud-stored images while enabling secure keyword-based search without exposing original content.

---

## 2. Secure QR Code Scanner to Detect Malicious URLs using Machine Learning

Description:  
A Machine Learning-based QR code scanner that identifies malicious URLs, phishing attacks, and suspicious web redirections.

Technologies Used:
- Python  
- Django Framework  
- Scikit-learn  
- Random Forest  
- SVM  
- KNN  
- MySQL  

Objective:
To protect users from phishing and malicious QR-based attacks through real-time classification models.
